# Ada Ad Creative Angles

Creative "ways in" - hooks and angles that make the ad interesting. Each populates the existing template (highlight → speak → suggestion → accept → CTA).

---

## Angle 1: The Cringe Moment
**Hook:** Show text everyone has written and immediately regretted.

| Field | Content |
|-------|---------|
| Tagline | `null` (let the cringe speak) |
| Editor Text | "I hope this email finds you {{well}}" |
| Transcription | "Ugh, everyone hates this" |
| Suggestion | "Hi Sarah," |
| CTA Description | "Kill your clichés" |

**Why it works:** Instant recognition. Viewer thinks "I've written that." Self-aware humor.

---

## Angle 2: The Jargon Call-Out
**Hook:** Corporate speak that everyone uses but nobody likes.

| Field | Content |
|-------|---------|
| Tagline | `null` |
| Editor Text | "Let's {{circle back}} on this next week" |
| Transcription | "Just say 'meet'" |

**Variants:**
- "We need to {{leverage synergies}}" → "work together"
- "Please {{advise}}" → "let me know"
- "{{Per my last email}}" → "As I mentioned"

## Angle 4: The Hedge

| Field | Content |

| Transcription | "Be direct" |

**Variants:**

- "I {{was wondering if maybe}}" → "Can you"
- "{{Sorry to bother you, but}}" → (delete)

## Angle 12: The Specificity Problem
**Hook:** Vague claims that mean nothing.

| Field | Content |
|-------|---------|
| Tagline | `null` |
| Editor Text | "Sales increased {{significantly}}" |
| Transcription | "How much?" |
| Suggestion | "by 34%" |
| CTA Description | "Be specific" |

**Variants:**
- "{{Many}} customers reported" → "How many?" → "73% of customers"
- "{{Recently}}" → "When?" → "Last quarter"
- "{{A leading company}}" → "Which one?" → "Salesforce"

## Production Priority

**High impact, universal recognition:**
1. Angle 2 - "Circle back" (everyone hates this phrase)
2. Angle 1 - "Hope this finds you well" (universal cringe)
3. Angle 12 - "Significantly" → specific number (clear value demo)
4. Angle 4 - Hedging language (relatable doubt)

**Strong demonstration of capability:**
5. Angle 6 - "Is 'love' too strong?" (shows Socratic mode perfectly)
6. Angle 5 - Date replacement (shows Didactic mode simply)
7. Angle 15 - "Just say we're canceling" (shows Ada handles hard conversations)
